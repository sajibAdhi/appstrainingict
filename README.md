গণপ্রজাতন্ত্রী বাংলাদেশ সরকারের তথ্য ও যোগাযোগ প্রযুক্তি বিভাগের উদ্যোগে ‘স্কিল ডেভেলপমেন্ট ফর মোবাইল গেইম এন্ড অ্যাপ্লিকেশান প্রোজেক্ট’ এর তত্ত্বাবধানে দেশব্যাপী বিনামূল্যে প্রশিক্ষণ প্রদানের জন্য ‘ট্রেনিং ফর মোবাইল অ্যাপ্লিকেশান ডেভেলপমেন্ট (Cross Platform)’ নামক একটি প্রকল্প গ্রহণ করা হয়েছে। প্রশিক্ষণটি বাংলাদেশের ৬৪ টি জেলায় শুরু হয়েছে। সরকারি এই প্রশিক্ষণটি সম্পন্ন বিনা খরচে করা যাবে।
